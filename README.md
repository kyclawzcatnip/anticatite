# 🐱 Super Cat World

A retro 2D platformer starring a brave cat on an adventure through worlds, castles, and sky islands!

## 🎮 How to Play

- **Space** — Start the game
- **2** — Start Co-Op mode
- **Arrow Keys / WASD** — Move & Jump
- Touch controls available on mobile

## 🚀 Play Online

Open `index.html` in your browser, or visit the live version on GitHub Pages (see below).

## 🌐 Deploying to GitHub Pages

1. Push this repository to GitHub.
2. Go to **Settings → Pages**.
3. Under **Source**, select **Deploy from a branch**.
4. Choose the **main** branch and **/ (root)** folder.
5. Click **Save** — your game will be live at `https://<your-username>.github.io/anticatite/`.

## 📁 Project Structure

```
index.html   — Game page
style.css    — Styling & layout
game.js      — All game logic & rendering
```
